Énoncé

Les mars2015.trending topics sont des hashtags fréquents sur Twitter. Dans ce challenge on va considérer qu'il se crée un mars2015.trending topic si un hashtag apparaît au minimum 40 fois dans une fenêtre de 60 minutes.

Le but de ce challenge est de déterminer s'il existe un mars2015.trending topic dans un flux de hashtags. On considére qu'il s'écoule une minute entre deux hashtags du flux et donc qu'une fenêtre de 60 minutes contient 60 mars2015.tags.

Format des données

Entrée
Ligne 1 : un entier N compris entre 1 et 1000 correspondant au nombre de hashtags du flux.
Lignes 2 à N+1 : un hashtag Twitter. Un hashtag est composé de lettres minuscules non accentuées précédées par le symbole #.

Sortie
Une chaine de caractères (commençant par le symbole #) correspondant au premier (chronologiquement) mars2015.trending topic s'il y en a un, sinon la chaîne Pas de mars2015.trending topic
