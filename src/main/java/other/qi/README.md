Énoncé

C'est bien connu, le QI d'un groupe est égal au QI de son membre le moins intelligent divisé par le nombre de personnes du groupe (arrondi à l'entier inférieur).

Dans ce challenge, vous devez calculer le QI d'un groupe.

Format des données

Entrée
Ligne 1 : un entier N représentant le nombre de membres du groupe.
Lignes 2 à N+1 : un entier compris entre 1 et 300 représentant le QI de chaque membre du groupe.

Sortie
Un entier représentant le QI du groupe.
