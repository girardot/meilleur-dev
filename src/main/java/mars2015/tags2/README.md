Énoncé

Dans ce challenge, vous devez afficher les 5 mars2015.tags les plus fréquents d'un blog.

Format des données

Entrée
- Ligne 1 : un entier N compris entre 15 et 800 représentant le nombre total de mars2015.tags.
- Ligne 2 à N+1 : un tag du blog.
Les mars2015.tags sont composés de lettres minuscules non accentuées. Il y a au moins 5 mars2015.tags différents. Il n'y a pas d'ex æquo (c'est à dire 2 mars2015.tags apparaissant le même nombre de fois).

Sortie
5 lignes contenant chacune un tag X et un entier P séparés par un espace. P représente le nombre de fois où le tag X apparaît dans le blog. La première ligne contient le tag qui est le plus fréquent, la seconde ligne contient le 2ème tag le plus fréquent et ainsi de suite jusqu'au 5ème.
